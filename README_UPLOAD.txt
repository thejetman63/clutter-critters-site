   Search for: [Your City / Area]

CONTACT FORM
- Default behavior: opens an email draft on submit.
- For a real submit endpoint: set the <form action="..."> to Formspree (or similar) and remove "REPLACE_ME".

GODADDY UPLOAD (cPanel / File Manager)
1) In GoDaddy, open Hosting > cPanel (or File Manager).
2) Go to the website root folder (usually public_html).
3) Upload EVERYTHING in this folder (index.html, styles.css, etc.).
4) Make sure index.html is in public_html (not inside another folder).
5) Visit your domain to verify.

If your domain shows an old page, clear browser cache or check if GoDaddy has a different default (like index.php).
